## Architecture Overview

This directory contains architecture diagrams for the Wazuh SOC lab.

- wazuh-soc-architecture.png  
  Shows the logical flow of security events from Linux agents to the Wazuh dashboard.

- network-architecture.png  
  Shows the VirtualBox network design (NAT + Host-only) used to ensure
  internet access and secure host-to-VM communication.

These diagrams reflect the actual deployed lab, not theoretical designs.
